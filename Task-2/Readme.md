# 🤖 Basic Chatbot

A simple rule-based chatbot developed using **Python**. The chatbot interacts with users by responding to predefined inputs using conditional statements. This project is designed to demonstrate the fundamentals of chatbot development and Python programming.

## 📌 Features

- Greets the user with a welcome message.
- Responds to basic inputs such as:
  - Hello
  - How are you?
  - Bye
- Continues the conversation until the user enters **"bye"**.
- Displays a default message for unknown inputs.

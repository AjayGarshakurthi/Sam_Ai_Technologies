# 📌 Project Overview

This project is a simple command-line calculator developed using Python. It allows users to perform basic arithmetic operations by entering two numbers and selecting an operation. The calculator then processes the input and displays the result.

# ✨ Features
Addition
Subtraction
Multiplication
Division
User-friendly input prompts
Handles division by zero
Displays an error message for invalid operations

# 🛠️ Technologies Used
Python 3
Visual Studio Code (VS Code)

# 📂 How to Run
Install Python 3 on your system.
Download or clone this repository.
Open the project in VS Code or any Python IDE.

# Run the program:

python calculator.py
Enter two numbers.
Choose an arithmetic operation (+, -, *, /).
View the calculated result.

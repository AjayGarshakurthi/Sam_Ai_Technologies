# Sam_Ai_Technologies - Python Programming

## 📖 Overview

This repository contains three Python projects developed to strengthen programming fundamentals, problem-solving skills, and AI integration. The projects range from a simple calculator to an AI-powered voice assistant, showcasing practical applications of Python.

## 📂 Projects Included

### 🧮 1. Basic Calculator
A command-line calculator that performs basic arithmetic operations.

**Features**
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly menu
- Input validation
- Division by zero handling

---

### 🤖 2. Basic Chatbot
A simple rule-based chatbot that interacts with users through the command line and responds to predefined questions.

**Features**
- Greeting responses
- Interactive conversation
- Answers common questions
- Beginner-friendly implementation
- Easy-to-understand Python logic

---

### 🎙️ 3. Python AI Voice Assistant
An AI-powered voice assistant that listens to voice commands, converts speech to text, processes queries using an AI language model, and responds with voice output.

**Features**
- Speech Recognition
- Text-to-Speech
- AI-powered Question Answering
- Voice Commands
- Continuous Listening
- Natural Language Processing
- API Integration

---

# ▶️ Running the Projects

### Basic Calculator

```bash
cd Basic_Calculator
python calculator.py
```

### Basic Chatbot

```bash
cd Basic_Chatbot
python chatbot.py
```

### Python AI Voice Assistant

```bash
cd Python_AI_Voice_Assistant
python main.py
```

---

# 🔑 API Configuration (For AI Voice Assistant)

Create a file named **config.py** inside the `Python_AI_Voice_Assistant` folder.

```python
API_KEY = "YOUR_API_KEY"
```

> **Note:** Do not upload your API key to GitHub. Add `config.py` to `.gitignore` to keep it private.
